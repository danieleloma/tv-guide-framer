   ```bash

   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Test the conversion:**
   ```bash
   npx ts-node scripts/excel-to-guide.ts --in ./sample-data.csv --out ./public/guide.json --channelId "your-channel" --defaultRegion "ROA"
   ```

4. **Copy to Framer project:**
   - Copy the `components/` folder to your Framer project
   - Add dependencies to Framer's package.json
